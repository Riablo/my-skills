# 快递取件提醒

自动提取快递取件信息，添加到 Apple Reminders，确保不会错过取件时间。

## 前置要求

### 安装 remindctl

```bash
brew install steipete/tap/remindctl
remindctl authorize
```

授权后 remindctl 可以读写 Apple Reminders。首次使用会自动创建「快递取件」列表。

### 安装 OCR 支持（可选，用于图片识别）

```bash
brew install tesseract tesseract-lang
```

不安装也可以使用文字输入方式。

---

## 使用方式

直接在 Claude Code 对话中发送快递信息即可：

**文字方式**，粘贴短信或通知内容：

```
您有一件快递已到达菜鸟驿站，取件码 123456，请尽快取件
```

**图片方式**，发送取件码截图，AI 会识别并提取信息。

---

## 提醒时间规则

| 收到信息的时间 | 提醒时间 |
|--------------|----------|
| 00:00 – 11:00 | 当天 11:00 |
| 11:00 – 17:00 | 当天 17:00 |
| 17:00 – 24:00 | 次日 09:00 |

---

## 支持识别的信息

**取件码格式**
- 6–8 位纯数字
- 字母+数字组合（如 A123456）
- 带「取件码」前缀的格式

**存放地点**
- 菜鸟驿站、丰巢柜、快递柜、门卫、前台、超市代收点

**快递公司**
- 顺丰、中通、圆通、申通、韵达、百世、极兔、京东、邮政、EMS 等
